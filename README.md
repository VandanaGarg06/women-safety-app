# **Women's Safety Web App**  
*A real-time safety platform for emergency alerts, location tracking, and community-driven security*  

## **🌍 Overview**  
The **Women's Safety Web App** is a **full-stack emergency response platform** designed to provide immediate protection and support during crises. By integrating **real-time SOS alerts, live location tracking, emergency contact management, and community-based safety reviews**, the app empowers users to stay secure and connected.  

Built with **user privacy and rapid response** in mind, this project bridges technology and social responsibility to create a safer environment for individuals.  

---

## **✨ Key Features**  

### **🚨 Emergency SOS Alerts**  
- **One-touch SOS button** for instant emergency notifications.  
- **Automatically shares live location** with trusted contacts.  
- **Real-time alerts** via SMS/email for immediate assistance.  

### **📍 Live Location Tracking**  
- **Interactive maps** powered by **Leaflet.js**.  
- **Continuous GPS updates** for real-time movement tracking.  
- **Secure location history** (optional storage for past routes).  

### **🔐 Secure Authentication**  
- **JWT-based login** for encrypted sessions.  
- **Google OAuth integration** (via Clerk/Auth0).  
- **Role-based access control** (user/admin permissions).  

### **📂 Cloud-Based Media Storage**  
- **Profile/contact images** stored securely via **Cloudinary**.  
- **Fast retrieval** with CDN support.  

### **👥 Emergency Contacts Management**  
- **Add/remove trusted contacts** (limit: 3–5 for quick access).  
- **Multi-channel alerts** (SMS, email, in-app notifications).  

### **📝 Safety Reviews & Community Alerts**  
- **Rate locations** as *Safe/Unsafe* based on personal experience.  
- **View crowd-sourced safety data** before visiting new areas.  

### **🔔 Real-Time Notifications**  
- **Instant SOS alerts** to contacts.  
- **Push notifications** for critical updates.  

---

## **⚙️ How It Works**  

1. **Sign Up / Login**  
   - Secure authentication via **email, password, or Google**.  

2. **Set Up Profile**  
   - Upload a photo, add emergency contacts.  

3. **Emergency Mode**  
   - **Press SOS** → Alerts + live location sent to contacts.  

4. **Location Tracking**  
   - Real-time GPS updates on an **interactive map**.  

5. **Review Locations**  
   - Share safety feedback for public places.  

6. **Manage Settings**  
   - Update profile, contacts, or privacy preferences.

## **🛠 Tech Stack**  
**Frontend:** React.js, Leaflet.js (Maps), Tailwind CSS  
**Backend:** Node.js, Express  
**Database:** MongoDB / PostgreSQL  
**Auth:** JWT, Clerk/Auth0  
**Storage:** Cloudinary (Images), Multer (Uploads)  
**Real-Time:** WebSockets, Firebase (Notifications)
